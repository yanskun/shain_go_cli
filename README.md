# 社員登録 CLI

```
1. 登録
2. 参照
3. 更新 無くてもいいかも
4. 削除
5. 終了
```


## 用件
- 登録情報: 名前、誕生日、性別、給与。登録後にIDが自動生成される。
- ちゃんとバリデーションする。
